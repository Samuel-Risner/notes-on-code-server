# Extensions

- For compiling: `formulahendry.code-runner`
- For linting: `llvm-vs-code-extensions.vscode-clangd`

You might have to change the extension settings in `llvm-vs-code-extensions.vscode-clangd`.

# Dockerfile

gcc -v
g++ -v
